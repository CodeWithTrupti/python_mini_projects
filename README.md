 🐍 Python Mini Projects  

This repository contains a collection of *beginner-friendly* Python projects that cover basic concepts like *logic building, user input handling, and simple algorithms*.  

📌 Projects Included  
1. 🖩 Calculator – A simple arithmetic calculator for basic operations (+, -, *, /).  
2. 🔑 Password Generator – Generates strong, random passwords with customizable length and character sets.  
3. 🪨📄✂ Stone Paper Scissors – A classic game where you play against the computer.  
4. ✅ To-Do List – A simple CLI-based task manager to add, remove, and view tasks.

Run any project using : python project_name. For example, to run the calculator: python calculator.py

🛠 Technologies Used
Python (Core Concepts, Random Module, Input Handling)
Basic File Handling (for To-Do List)
